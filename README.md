## 1、基础知识
- 缩进、代码块构建：使用缩进、不用大括号

- 变量、赋值：不是容器、标签
- 表达式：// 整除
- 字符串：`"abc"`、`'abc'`、`"""abc"""`
- 数值：
  - 整数可为任意大小
  - 内置处理函数、高级math、复数计算、高级复数cmath
  - [scipy](https://scipy.org/) Python中科学计算的基本算法 NumPy(执行密集型数值计算)
- None
- input 输入
- 操作符
- 编码风格：Python增强提案8​（Python Enhancement Proposal 8，PEP 8）
